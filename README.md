# vue-sample

For studying Vue.js

## 検証で使用した環境

- Vue.js 3.0.11
- Vue CLI 4.5.13
- Yarn 1.22.5
- node 16.1.0

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
